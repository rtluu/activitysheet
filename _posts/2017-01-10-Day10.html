Gulp is a javascript task runner that helps automate tasks like bundling code and minifying images/gifs etc.

Because my personal site will have a lot of heavy imagery, I decided to use gulp to build my personal site so that it's fast and snappy.

I like it's plugin/modular infrastructure and the fact that it uses sass by default.

On another note, the site design continues to change as I show people and get feedback. I'm think I'm goint to adjust my user flow so that there are individual pages for each project.

It's cool to see how it changes day over day.
